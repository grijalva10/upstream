# Reference Data

Static reference data used by agents and orchestrator.

## Folders

| Folder | Purpose |
|--------|---------|
| `costar/` | CoStar API lookups, market data, payload documentation |

## Adding New Reference Data

Create a new subfolder for each data source or domain:
```
reference/
├── costar/          # CoStar API
├── email-templates/ # (future) Outreach templates
└── ...
```
